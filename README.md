## 🚀 Live Demo

Open `index.html` in any modern browser — **no installation required!**

## 🎯 Features

| Feature | Description |
|---------|-------------|
| 🔥 **Neuron Heatmaps** | Color-coded activation maps for each hidden layer |
| 🌐 **Concept Clusters** | Visual 2D map showing how concepts group in latent space |
| 🔗 **Hidden Layer Viz** | Interactive network architecture with live activation glow |
| 📊 **Prediction Bars** | Real-time probability distribution for all output classes |
| 📋 **Inference Log** | Step-by-step processing timeline |
| 📥 **Export Report** | Download a full text-based visualization report |

## 🧠 How It Works

1. **Select an input sample** (image, text, tabular, or medical data)
2. **Choose a neural architecture** (CNN, FFN, RNN, or Transformer)
3. **Click "Run Inference"** to watch neurons activate layer by layer
4. **Explore the heatmaps** by switching between layers
5. **Export** the report for documentation
